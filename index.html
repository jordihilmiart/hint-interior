Halaman Utama

Ini adalah contoh halaman index.html yang siap diunggah ke GitHub. Anda dapat memodifikasinya sesuai kebutuhan.

Cara Menggunakan:

  1. Salin kode ini ke index.html di komputer Anda.
  2. Buat repositori baru di GitHub.
  3. Unggah file index.html ke repositori tersebut.
  4. Klik Settings → Pages → Pilih Main branch → Simpan.
  5. Halaman akan secara otomatis diterjemahkan menjadi: https://username.github.io/nama-repo/
